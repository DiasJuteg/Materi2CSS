    Jurusan Rpl

    Profil Jurusan PPLG

    Progam pengembangan perangkat lunak dan GIM atau sebelumnya keahlian Rekayasa Perangkat Lunak (RPL) adalah salah satu kopetensi keahlian dalam bidang Teknologi Komputer informatika secara khusus mempelajari tentang pemograman komputer. Lulusan RPL dewasa ini sangat dibutuhkan untuk menjawab benyaknya kebutuhan industri bagi ketersediaan tenaga Teknisi dalam bidang pengembangan software. Beberapa ruang lingkup pekerjaan di bidang RPL adalah Software developer, Game developer, Software tester, Software engineering, Software analist dan integrator konsultan IT dan Progammer.

    Peluang karir jurusan RPL bisa melingkupi, Software engineer/progammer, Mobile computing developer, IT consultant, Sistem analyst, Game developer, Software tester dan masih banyak lagi prospek karir lainnya.
